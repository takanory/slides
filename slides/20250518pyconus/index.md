## **sphinx-nekochan**

### **sphinx-nekochan**

* A Sphinx extension for adding the **Nekochan(cat) emoji** to documents
* {fas}`globe` [sphinx-nekochan.readthedocs.io](https://sphinx-nekochan.readthedocs.io/)

### Get Started

```bash
(env) % pip install sphinx sphinx-nekochan
(env) % sphinx-quickstart
```

```{code-block} python
:caption: conf.py

extensions = [
    ...
    "sphinx_nekochan",
]
```
